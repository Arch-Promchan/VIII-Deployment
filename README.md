# Exercise VIII - DevOps Deployment

[![Coverage Status](https://coveralls.io/repos/github/Arch-Promchan/VIII-Deployment/badge.svg?branch=main)](https://coveralls.io/github/Arch-Promchan/VIII-Deployment?branch=main)

## Description
This project implements a CI pipeline for a JavaScript utility library using **GitHub Actions** and **Coveralls**. It demonstrates automated testing, code coverage reporting, and bug fixing practices.

## Features
* **Unit Testing:** Implemented 30 unit tests using Jest with 76% coverage.
* **CI Pipeline:** Automated testing via GitHub Actions on every push.
* **Coverage Reporting:** Integrated with Coveralls.io for real-time coverage visualization.
* **Bug Fixes:** Identified and fixed critical logic errors in `clamp.js`, `divide.js`, `eq.js`, and others.

## How to Run
1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Run tests:**
    ```bash
    npm test
    ```
